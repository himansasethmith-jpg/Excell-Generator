How to use:

  1. Select the number of rows you want to generate
  2. Choose the data type for the sample data
  3. Click "Generate Data" to create the data table
  4. Click "Export to Excel" to download the data as an XLSX file
  5. Open the downloaded file in Excel or any spreadsheet application
